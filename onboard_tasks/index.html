2. p, br 태그 작성

p, br

Hello world

p 태그는 단락을 나타내며, 단락의 시작과 끝을 나타냅니다.

br 태그는 줄바꿈을 나타내며, 단락의 내용을 나누지 않습니다.

3. a 태그 작성

a

 Hello world
Hello world with new tab

a 태그는 링크를 할당할 수 있는 태그입니다.
예를 들면 a href="http://www.naver.com" 은 네이버로 이동하는 링크를 할당하는 것입니다.

a href="#{id}" 를 이용하면 현재 페이지 내에서 해당 id를 가진 태그로 화면을 이동시키도록 할 수 있습니다.

이를 통해, a href는 외부의 웹 페이지 뿐 아니라, 현재 페이지 내에서도 동적으로 화면을 이동시킬 수 있음을 알 수 있습니다.

a 태그에서 target 옵션에 _blank를 주면 새 탭에서 링크를 열 수 있습니다.

4. img 태그 작성

img

img 태그는 이미지를 삽입할 수 있는 태그입니다.
src="이미지 주소" 를 통해 이미지를 삽입할 수 있습니다.

width, height를 통해 이미지의 크기를 조절할 수 있습니다.

alt를 통해 이미지가 로드되지 않았을 때, 대체 텍스트를 지정할 수 있습니다.

좌측에 넣은 이미지처럼, a 태그와 함께 사용하여 이미지를 클릭했을 때, 해당 이미지의 링크로 이동하도록 할 수 있습니다.

5. table 태그 작성

table
th 1 th 2
td 1 td 2

table 태그는 표를 나타내는 태그입니다.
tr 태그는 표의 행을 나타내며, td 태그는 표의 열을 나타냅니다.
th 태그는 표의 열의 제목을 나타냅니다.
